# **Pascal Roguelike**

## Movement

 - **W** - *up*
 - **A** - *left*
 - **S** - *down*
 - **D** - *right*

____

## Combat

 - **E** - *melee combat*
 - **Q** - *range combat*

____

## Interface

 - **HP** - *health points*
 - **Money** - *money*
 - **Inv** - *inventory*
 - **M**/**R** - *melee*/*range*
 - **S** - *strength*
 - **D** - *distance*

____

## Symbols

 - **@** - *main character*
 - **N** - *npc*
 - **?** - *loot*
 - **$** - *shop*
 - **D**/**H** - *demented*/*hunter*
 - **#** - *path*
 - **.** - *ground*
 - **+** - *door*
 - **-**/**|** - *walls*

____

### Side effect

since pascal cannot restore  
the color of the text back,  
you will have to use the  

```bash
reset
```
command to restore the color text in the terminal
